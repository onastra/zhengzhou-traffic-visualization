# 郑州市交通优化可视化系统

这是一个用于可视化和优化郑州市交通的 Web 应用程序。

## 如何使用

1. 访问我们的网站：[https://onastra.github.io/zhengzhou-traffic-visualization/]

2. 在首页上，您可以：
   - 查看郑州市的交通地图
   - 规划路线
   - 查看交通流量分析
   - 探索交通知识图谱

3. 要规划路线，只需在输入框中输入起点和终点，然后点击"规划路线"按钮。

4. 交通流量分析和知识图谱会自动更新。

## 反馈
